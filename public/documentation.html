myFlix Documentation

This API will be a place where users can access information about different movies, directors, and genres.

Users will be able to sign up, update their personal info, and create a list of their favorite movies.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of ALL movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie, containing description, genre, director, image URL, and whether it is featured or not.
Return data about genre by name /genre/:name GET None A JSON object holding data about a genre and it's description
Return data about a director by name /directors/:name GET None A JSON object holding data about a single director, containing a bio, birth year, and death year
Allow new users to register /users POST A JSON object holding data about a user to add, structured like:
{
name: "John Doe"
email: "jd1234@email.com"
username: "JD1234"
password: "whoAmI?"
}		 A message indicating whether the registration was successful
Allow users to update their info /users/:username PUT A JSON object holding data about a user to add, structured like:
{
name: "John Doe"
email: "jd1234@email.com"
username: "JD1234"
password: "whoAmI?"
}		 A message indicating whether the update was successful
Allow users to deregister /users/:username DELETE None A message indicating whether the user was deregistered
